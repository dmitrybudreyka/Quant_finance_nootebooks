# Quant Finance Notebooks

This repository contains my personal notebooks related to quantitative finance, financial mathematics, trading, risk modelling, and data analysis.

The goal of this repository is to document my learning process and build a portfolio of practical quant finance projects using Python.

## Topics

Projects in this repository may cover topics such as:

- Option pricing
- Black-Scholes model
- Monte Carlo simulations
- Portfolio optimization
- Risk measures such as VaR and CVaR
- Time series analysis
- Market data analysis
- Trading strategy backtesting
- Stochastic processes
- Numerical methods in finance

## Current Projects

### Geometric Brownian Motion Interactive Notebook

Notebook: `geom_brownian_interactive.ipynb`

This notebook explores the Geometric Brownian Motion model, which is commonly used to model stock price dynamics in quantitative finance.

Main ideas covered:

- Simulation of stochastic price paths
- Influence of drift and volatility
- Interactive visualization of simulated asset prices
- Connection to the Black-Scholes framework

### Financial Data Import and Visualisation Notebook

Notebook: `fin_data_import_and_visualisation.ipynb`

This notebook introduces basic financial data analysis using `yfinance`, pandas, and matplotlib. It downloads historical stock and ETF prices, plots price series, converts prices into returns, and computes key performance and risk measures.

Main ideas covered:

- Importing market data from Yahoo Finance
- Visualising stock and ETF price series
- Computing compound returns
- Annualizing returns and volatility
- Calculating a raw Sharpe ratio
- Building a wealth index and analysing drawdowns
